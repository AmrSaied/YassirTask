# YassirTask

A simple UIKit app that displays movies using TheMoviesDB APIs.

# Screens 

* Movies list screen.
* Movie details screen.

# Technologies

* UIKit
* RXSwift/RXCocoa
* MVVM
* CoreData
* Unit Testing

# Development Target Pods 

* RxSwift/RxCocoa

# Testing Target Pods

* RxTest

# Requirements

* iOS 16 or later.
* Xcode 14.3 or later.
